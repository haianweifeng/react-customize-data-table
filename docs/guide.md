## Install

`npm i react-customize-data-table`

## Development

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father`,

```bash
$ npm run compile
```

## Usage

```js
import Table from 'react-customize-data-table';

ReactDOM.render(<Table />, container);
```
