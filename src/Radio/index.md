### API

| 属性      | 类型                                   | 默认值  | 说明             |
| --------- | -------------------------------------- | ------- | ---------------- |
| className | string                                 | -       | 扩展 className   |
| style     | CSSProperties                          | -       | 样式对象         |
| checked   | boolean                                | 'false' | 勾选状态         |
| disabled  | boolean                                | 'false' | 开启禁用状态     |
| onChange  | (checked: boolean, event: any) => void | -       | 变化时的回调函数 |
