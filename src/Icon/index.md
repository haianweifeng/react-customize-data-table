### API

| 属性      | 类型          | 默认值 | 说明                             |
| --------- | ------------- | ------ | -------------------------------- |
| className | string        | -      | 图标样式名                       |
| style     | CSSProperties | -      | 图标样式对象                     |
| component | ComponentType | -      | 渲染根标签为 `svg` 的 React 组件 |
